Web GIS and Interactive Files

Below are a list of Web GIS and Interactive File examples that Natural Capital Solutions offer.

Web Mapping Portals


Web Mapping Portals provide a platform to view datasets in an interactive form, making it easier for users to investigate and analyse data. Portals can include additional tools such as: layer swipes, location lookup, print, data edits.
Example

ArcGIS StoryMaps


Immersive and engaging stories using ArcGIS StoryMaps. Lead users through the journey of your project using spatial datasets and dynamic, scrollable content.
Example

Html File Data Viewers


A lightweight version of the Web Mapping Portals. These file data viewers display an interactive map for users to investigate and analyse data, without the need for desktop GIS software, whilst being bundled into a single file that is easy to share with others.
⚠ There are limits on the size of datasets displayed in the data viewers due to their lightweight structure. Farms, estates, and other small study areas are suitable for this output. Large study areas should consider the Web Mapping Portal option above.

Interactive Html Reports


Alongside a standard pdf format, natural capital assessment reports can also be delivered in html format. Html reports appear similar to pdf reports, but have the benefit of interactive content, such as maps and data tables. This allows users to zoom, pan, and alter map layers all within the report file - providing improved viewing of datasets compared to a static image found in the pdf formats, and removing the need for desktop GIS software to view datasets.
